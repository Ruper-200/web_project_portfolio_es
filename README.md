📄 README – Prueba de Web Responsiva con IA
🧩 Descripción

Este proyecto es una práctica de maquetación web responsiva basada en un diseño tipo portafolio. El objetivo fue aprender a estructurar correctamente un layout usando HTML y CSS, así como adaptar el diseño a diferentes tamaños de pantalla (desktop, tablet y móvil).

🛠️ Herramientas utilizadas
HTML5
Para la estructura del contenido (etiquetas semánticas como header, section, nav, etc.)
CSS3
Para estilos, distribución del layout y efectos visuales.
CSS Grid
Utilizado para organizar los elementos principales en columnas y filas.
Flexbox
Usado en algunos componentes como el menú de navegación.
Media Queries
Para hacer el diseño responsivo en diferentes dispositivos.
🎯 Objetivos de aprendizaje
Entender la diferencia entre Grid y Flexbox
Practicar la maquetación responsiva
Aprender a usar position para elementos decorativos (como la luz)
Mejorar la organización del código CSS
Aplicar buenas prácticas básicas de estructura HTML
📱 Responsive Design

El diseño se adapta a 3 tamaños principales:

Desktop (>1024px) → Layout en dos columnas
Tablet (768px – 1023px) → Elementos centrados, imagen arriba
Mobile (<768px) → Layout en una sola columna
✨ Características
Header fijo en la parte superior
Imagen de perfil con efecto de luz (usando radial-gradient)
Botones con efecto hover
Navegación simple con animación en enlaces
